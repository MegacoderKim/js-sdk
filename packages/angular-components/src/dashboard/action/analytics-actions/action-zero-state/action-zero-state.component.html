Use actions to build your use case

An Action represents a pickup, delivery, appointment, drop-off, visit, stopover, or any other transaction event being performed by your Users. Use actions to build a use case of your choice, e.g., Order Tracking, Mileage Tracking, and Live Location Sharing,.

You need to set up billing to use Actions. Check pricing for details. In case you want to play with our APIs before putting in your card, switch to the Test environment. Once you are comfortable, switch back to Production environment.

Once you set up billing, start using Actions from the platform/language of your choice.
